# Shared Workflow Utilities

Common utilities shared across all workflow implementations.

## Overview

The `workflows/shared/` module provides common state management, subagent spawning, and file I/O utilities used by deep-interview, ralplan, team, and ultragoal workflows.

## Key Utilities

- State file read/write helpers
- Workflow phase management
- Common type definitions
