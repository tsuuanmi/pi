# Harness Runtime

Runtime environment for the coding agent harness, providing sandboxing, file system access, and shell execution.

## Overview

The harness runtime provides the execution environment for the coding agent, including:

- File system operations (read, write, list, stat)
- Shell command execution with output capture
- Environment variable access
- Working directory management
