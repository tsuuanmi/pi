# Spec: count_words Tool

## Goal
Add a `count_words` tool to the coding agent that counts the number of words in a given text or file.

## Acceptance Criteria
- [ ] Tool `count_words` is registered.
- [ ] Correctly counts words in a string.
- [ ] Correctly counts words in a file.
- [ ] Handles empty files/strings gracefully.

## Technical Context
- Tool should be implemented as a standard AgentTool.
- Word counting should be based on whitespace separation.
